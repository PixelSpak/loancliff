# LoanCliff Design System

Generated by `/plan-design-review` on 2026-04-29. This is the source of truth for visual and interaction decisions.

## Brand
One number. The gap. Everything else is scaffolding.
The red dollar figure is the product. Every other design decision either supports it or gets out of its way.

The only accent color is red. No blues, purples, or greens as brand colors.
Green is reserved for the "cap" row in breakdown tables (it signals "what federal aid covers").
Blue is reserved for hyperlinks only.

## Interaction States

| Element | State | Spec |
|---------|-------|------|
| Program select | School not selected | Disabled, `opacity-50 cursor-not-allowed` |
| Program select | School selected | Shows only programs for that school |
| School select | School changes | Program select resets to blank |
| CTA button | Loading/disabled | `opacity-50 cursor-not-allowed` when form incomplete |
| Share button | Clicked | Label changes to "Link copied!" for 1500ms, then reverts |
| Result section | Pre-submit | Hidden (`result === null`) |
| Result section | Post-submit | Scrolls into view (`ref.current?.scrollIntoView({ behavior: 'smooth' })`) |

## Copy Principles

- Direct address: "Your funding gap" not "The funding gap"
- Numbers first: "$208,000 total gap" before explaining what that means
- No hedging in headlines: "Here's your gap" not "Here's an estimate of your potential gap"
- Disclosure language is honest but brief: one sentence, caption size, not in the critical path
- FAQ answers embed the school-specific numbers (done ✓)

## Accessibility

- Keyboard nav: all interactive elements reachable via Tab
- Focus visible: `focus-visible:ring-2 focus-visible:ring-red-600 focus-visible:ring-offset-2` on primary CTA
- Skip link: `<a href="#results">Skip to results</a>` visually hidden, shown on :focus
- Touch targets: 44px minimum (py-3 on button ≈ 48px ✓, native selects ✓)
- Contrast: red-600 on white = 4.6:1 ✓ (WCAG AA). Red-500 on zinc-900 ≈ 4.8:1 ✓
- `aria-label` on breadcrumb nav ✓, on each section ✓
- Screen reader: currency values read correctly without special aria treatment

## What This Is Not

- Not a marketing landing page. No hero image. No 3-column feature grid. No testimonials.
- Not a dashboard. No sidebar. No data table with 10+ rows.
- It is: a **tool**. Form → number → options. The number is the product.
